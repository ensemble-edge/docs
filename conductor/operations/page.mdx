# page Operation

**Render full-stack React/htmx pages with SSR and dynamic routing.**

## Basic Usage

```yaml
operations:
  - name: dashboard
    operation: page
    config:
      component: Dashboard
      props:
        user: ${fetch-user.output}
        data: ${fetch-data.output}
```

## Configuration

```yaml
config:
  component: string    # React component name
  props: object       # Component props
  renderMode: string  # ssr, csr, or hybrid
```

## Examples

### SSR Page

```yaml
operations:
  - name: dashboard
    operation: page
    config:
      component: Dashboard
      renderMode: ssr
      props:
        user: ${user.output}
```

### Dynamic Routes

```yaml
operations:
  - name: product-page
    operation: page
    config:
      component: ProductPage
      props:
        product: ${fetch-product.output}
        reviews: ${fetch-reviews.output}
```
