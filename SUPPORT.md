# Support

## Getting Help

If you need help with Ensemble documentation, there are several ways to get support:

### Documentation

- **[Documentation Site](https://docs.ensemble.ai)** - Browse the full documentation
- **[Getting Started](https://docs.ensemble.ai/getting-started)** - Quick start guides
- **[GitHub](https://github.com/ensemble-edge)** - Source repositories

### Community Support

- **[GitHub Discussions](https://github.com/orgs/ensemble-edge/discussions)** - Ask questions and discuss
- **[GitHub Issues](https://github.com/ensemble-edge/docs/issues)** - Report documentation issues

### Contact

- **Email**: [hello@ensemble.ai](mailto:hello@ensemble.ai)

## Reporting Documentation Issues

Found a typo, broken link, or outdated content? Please:

1. **Check if it's already reported** - Search existing issues
2. **Open an issue** - Describe what's wrong and where
3. **Submit a PR** - Even better, fix it yourself!

## Contributing to Documentation

We welcome documentation contributions! See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines on:

- Writing and formatting documentation
- Submitting pull requests
- Documentation standards

## Reporting Security Issues

**Do not open public GitHub issues for security vulnerabilities.**

Please report security issues privately to [security@ensemble.ai](mailto:security@ensemble.ai) or see our [Security Policy](SECURITY.md).
