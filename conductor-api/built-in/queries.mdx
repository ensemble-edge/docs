## Overview

The `queries` built-in member executes SQL queries using Cloudflare Hyperdrive for connection pooling and query optimization.

```yaml
- member: fetch-users
  type: Function
  config:
    builtin: queries
    query: SELECT * FROM users WHERE status = ?
    params: [active]
    catalog: main
```

## Examples

### Simple Query

```yaml
- member: get-user
  type: Function
  config:
    builtin: queries
    query: SELECT * FROM users WHERE id = ?
    params: [${input.userId}]
```

### Prepared Statement

```yaml
- member: insert-record
  type: Function
  config:
    builtin: queries
    query: INSERT INTO logs (event, data) VALUES (?, ?)
    params: [${input.event}, ${input.data}]
```

### Transaction

```yaml
- member: transfer-funds
  type: Function
  config:
    builtin: queries
    queries:
      - query: UPDATE accounts SET balance = balance - ? WHERE id = ?
        params: [${input.amount}, ${input.fromId}]
      - query: UPDATE accounts SET balance = balance + ? WHERE id = ?
        params: [${input.amount}, ${input.toId}]
    transaction: true
```
