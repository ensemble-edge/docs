## Overview

The `fetch` built-in member makes HTTP requests with built-in retry, timeout, and response parsing.

```yaml
- member: api-call
  type: Function
  config:
    builtin: fetch
    url: https://api.example.com/data
    method: GET
    headers:
      Authorization: Bearer ${env.API_KEY}
```

## Examples

### GET Request

```yaml
- member: fetch-data
  type: Function
  config:
    builtin: fetch
    url: ${input.apiUrl}
    method: GET
```

### POST Request

```yaml
- member: post-data
  type: Function
  config:
    builtin: fetch
    url: ${env.API_URL}
    method: POST
    headers:
      Content-Type: application/json
    body:
      data: ${input.data}
```

### With Retry

```yaml
- member: resilient-fetch
  type: Function
  config:
    builtin: fetch
    url: ${input.url}
  retry:
    maxAttempts: 3
    backoff: exponential
```
