## Overview

Edgit integration enables loading ensemble components from distributed registries.

```typescript
import { loadComponent } from '@ensemble-edge/conductor/sdk';

const config = await loadComponent('user-onboarding@1.0.0', env);
```

## Functions

### loadComponent()

Load component by reference

### loadMemberConfig()

Load member configuration

## Version References

Reference format: `name@version`
