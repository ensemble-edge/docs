## List Schedules

<api>GET /api/v1/schedules</api>

List all configured cron schedules.

### Response

<ResponseField name="schedules" type="array">
  Array of schedule objects

  <ResponseField name="schedules[].name" type="string">
    Schedule name
  </ResponseField>

  <ResponseField name="schedules[].ensemble" type="string">
    Ensemble to execute
  </ResponseField>

  <ResponseField name="schedules[].cron" type="string">
    Cron expression
  </ResponseField>

  <ResponseField name="schedules[].enabled" type="boolean">
    Whether schedule is active
  </ResponseField>

  <ResponseField name="schedules[].lastRun" type="number">
    Last execution timestamp
  </ResponseField>

  <ResponseField name="schedules[].nextRun" type="number">
    Next scheduled execution
  </ResponseField>
</ResponseField>

<RequestExample>

```bash cURL
curl https://your-worker.workers.dev/api/v1/schedules \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://your-worker.workers.dev/api/v1/schedules',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const { schedules } = await response.json();
schedules.forEach(s => {
  console.log(`${s.name}: ${s.cron} -> ${s.ensemble}`);
});
```

</RequestExample>

<ResponseExample>

```json Success (200)
{
  "schedules": [
    {
      "name": "daily-report",
      "ensemble": "generate-daily-report",
      "cron": "0 9 * * *",
      "enabled": true,
      "lastRun": 1704009600000,
      "nextRun": 1704096000000
    },
    {
      "name": "hourly-sync",
      "ensemble": "sync-data",
      "cron": "0 * * * *",
      "enabled": true,
      "lastRun": 1704067200000,
      "nextRun": 1704070800000
    }
  ]
}
```

</ResponseExample>

## Trigger Schedule Manually

<api>POST /api/v1/schedules/:name/trigger</api>

Manually trigger a scheduled ensemble (bypass cron timing).

<ParamField path="name" type="string" required>
  Schedule name
</ParamField>

<ParamField body="input" type="object">
  Optional input data to override defaults
</ParamField>

### Response

<ResponseField name="id" type="string">
  Execution ID
</ResponseField>

<ResponseField name="schedule" type="string">
  Schedule name
</ResponseField>

<ResponseField name="ensemble" type="string">
  Ensemble executed
</ResponseField>

<ResponseField name="timestamp" type="number">
  Execution timestamp
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://your-worker.workers.dev/api/v1/schedules/daily-report/trigger \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "input": {
      "date": "2024-01-01"
    }
  }'
```

```javascript JavaScript
const response = await fetch(
  'https://your-worker.workers.dev/api/v1/schedules/daily-report/trigger',
  {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      input: {
        date: '2024-01-01'
      }
    })
  }
);

const result = await response.json();
console.log(`Triggered execution: ${result.id}`);
```

</RequestExample>

<ResponseExample>

```json Success (200)
{
  "id": "exec_abc123def456",
  "schedule": "daily-report",
  "ensemble": "generate-daily-report",
  "timestamp": 1704067200000
}
```

```json Not Found (404)
{
  "error": {
    "code": "SCHEDULE_NOT_FOUND",
    "message": "Schedule 'unknown-schedule' not found"
  }
}
```

</ResponseExample>

## Cron Expression Format

Conductor uses standard cron format (5 fields):

```
* * * * *
│ │ │ │ │
│ │ │ │ └─ Day of week (0-6, 0=Sunday)
│ │ │ └─── Month (1-12)
│ │ └───── Day of month (1-31)
│ └─────── Hour (0-23)
└───────── Minute (0-59)
```

### Examples

| Expression | Description |
|------------|-------------|
| `0 9 * * *` | Every day at 9:00 AM |
| `*/15 * * * *` | Every 15 minutes |
| `0 */2 * * *` | Every 2 hours |
| `0 0 * * 0` | Every Sunday at midnight |
| `0 9 * * 1-5` | Weekdays at 9:00 AM |
| `30 8 1 * *` | First day of month at 8:30 AM |

### Configuration

Configure in `wrangler.toml`:

```toml
[triggers]
crons = [
  "0 9 * * *",      # Daily at 9am
  "*/15 * * * *",   # Every 15 minutes
  "0 0 * * 0"       # Weekly on Sunday
]
```

Handle in worker:

```typescript
export default {
  async scheduled(event: ScheduledEvent, env: Env): Promise<void> {
    const conductor = new Conductor({ env });

    // Determine which ensemble to run based on cron
    const cron = event.cron;

    if (cron === '0 9 * * *') {
      await conductor.executeEnsemble('daily-report', {
        date: new Date().toISOString()
      });
    } else if (cron === '*/15 * * * *') {
      await conductor.executeEnsemble('frequent-check', {});
    }
  }
};
```
